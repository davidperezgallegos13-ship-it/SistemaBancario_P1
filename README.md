# Sistema Bancario - Parcial I
Desarrollo de un sistema de gestión de cuentas bancarias en C# utilizando .NET Framework 4.7.2 y metodologías DevOps.

## Especificaciones del Sistema
- **Encapsulamiento:** Control de saldos mediante métodos seguros de depósito y retiro.
- **Gestión de Excepciones:** Validación ante montos negativos o fondos insuficientes.
- **Pruebas Unitarias:** Cobertura de código con MSTest.
